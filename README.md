# 🎂 Birthday Wishing App

A beautiful, interactive birthday surprise app that reveals a personalised wish when a matching name is entered.

---

## ✨ Features

- **Secret portal** — only the right name unlocks the birthday surprise
- **Confetti animation** on every birthday reveal
- **Flip card** with a personal message
- **Candle row** matching the person's age
- **Manage page** — add, update, or remove birthday people without touching code
- **Persists in browser** — names & messages saved via localStorage

---

## 🚀 How to Use

1. Open `index.html` in any modern browser (Chrome, Firefox, Edge, Safari)
2. Type one of the preset names and press **Open** or hit **Enter**

### Preset names (try these):
| Name    | Age |
|---------|-----|
| Priya   | 25  |
| Rahul   | 28  |
| Ananya  | 22  |
| Arjun   | 30  |
| Sneha   | 24  |

---

## ⚙️ Adding Your Own Names

Click the **⚙️** button (bottom-right corner) or open `manage.html` directly.

Fill in:
- **Name** — the name someone must type to unlock their surprise
- **Custom message** — a heartfelt birthday note (optional; a default is auto-generated)
- **Age** — used to display candles and a line like "Celebrating 25 incredible years!" (optional)

Hit **Save** — the name is live immediately!

---

## 📁 File Structure

```
birthday-app/
├── index.html    ← Main portal (enter a name here)
├── manage.html   ← Add / edit birthday people
├── style.css     ← All styles
├── app.js        ← All logic
└── README.md     ← This file
```

---

## 🌐 Deployment

Simply host the folder on any static file server:
- **GitHub Pages** — push to a repo and enable Pages
- **Netlify / Vercel** — drag & drop the folder
- **Local** — just double-click `index.html`

No server, no backend, no dependencies beyond an internet connection (for Google Fonts).
